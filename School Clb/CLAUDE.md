## 1. СОСТОЯНИЕ ПРОЕКТА (факт по коду, ~9 500 строк)

**Стек:** SwiftUI + (планируется) Firebase. Автор: Omar.

**Архитектура:** проект из **UIKit-шаблона**. `@main` на `AppDelegate`, SwiftUI грузится через `SceneDelegate` -> `UIHostingController(rootView: ContentView())`. Рабочая схема, не трогать без необходимости.

**Firebase НЕ подключён.** Ни один файл не импортирует Firebase, в `AppDelegate` нет `FirebaseApp.configure()`. Всё приложение на **mock-данных** (~21 файл с заглушками). Это полный UI-прототип без бэкенда.

**Дизайн-система (сохранять):** оранжевый `#E84E1B`, синий `#1B2A6B`, жёлтый `#FFE600`, светло-голубой `#F0F4FF`, фон белый. Шрифт SF Pro Rounded. Цвет через `Color(hex:)`.

**Навигация:** кастомный таб-бар, 5 табов: `Уроки` · `Движ` · `Брейк` · `Чат` · `Профиль`. (Движ -> переименовать в «Курс».)

**Карта файлов:**
- Root/навигация: `ContentView`, `AppRouter`, `MainTabView`, `AppDelegate`, `SceneDelegate`, `ViewController`
- Авторизация: `AuthView`, `AuthViewModel`, `User`
- Профиль: `ProfileView`, `ProfileViewModel`, `EditProfileView`, `EmojiPickerView`
- Уроки: `LessonsView`, `LessonsViewModel`, `CourseLevelsView`, `LessonListView`, `VideoLessonView`, `TestView`, `PracticeTestView`
- Брейк: `BreakView`, `BreakViewModel`, `DailyChallengeView`, `DuelView`, `QuizView`
- Движ/баллы: `MovementView`, `MovementViewModel`, `SpendPointsView`
- Чат/форум/ИИ: `ChatView`, `SOSViewModel`, `AskQuestionView`, `ForumChatView`, `AITeacherView`
- Премиум/оплата: `PremiumView`, `PaymentViewModel`, `PaymentWebView`, `DiscountView`
- Редактор/модерация: `EditorDashboardView`, `UploadLessonView`, `UploadTestView`, `ModerationView`
- Локализация: `Localizable.swift` (KG/RU)

**Тарифы в коде устарели:** `PaymentViewModel` = 299 сом/мес и 1999 до ОРТ. Нужны BASE/PRO/PROMAX А,Б,С (см. ниже).
