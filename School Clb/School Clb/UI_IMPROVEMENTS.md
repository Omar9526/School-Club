# Улучшение UI дизайна тарифных карточек

## Обзор изменений

Полностью обновлен дизайн экрана выбора тарифов (PremiumView) с современными визуальными эффектами, которые отлично работают как в светлой, так и в темной теме.

---

## 🎨 Новые визуальные эффекты

### 1. Заголовок экрана (Header)
**Улучшения:**
- ✨ Радиальный градиент для свечения вокруг иконки
- 🌟 Тень с цветом золота для эффекта "сияния"
- 👑 Градиент на иконке короны для объема
- 📏 Улучшенные отступы и иерархия текста

**Код:**
```swift
Circle()
    .fill(
        RadialGradient(
            colors: [
                Color(hex: "#FFE600").opacity(0.3),
                Color(hex: "#E84E1B").opacity(0.2),
                Color.clear
            ],
            center: .center,
            startRadius: 30,
            endRadius: 70
        )
    )
```

### 2. Карточки тарифов (TierCard)
**Полностью переработанный дизайн:**

#### Заголовок карточки:
- 🎯 Иконка в круге с градиентным фоном
- 📱 Улучшенная типографика (22pt bold для названия)
- 🏷️ Бейдж "ДЛЯ СТУДЕНТОВ" в правом верхнем углу с градиентом

#### Разделитель:
- 🌈 Градиентная линия вместо обычного Divider
- 🎨 Использует цвет тарифа для идентичности

#### Список функций:
- ✅ Увеличенные иконки (16pt вместо 14pt)
- 📝 Улучшенный интервал (12pt между элементами)
- 💪 Medium weight для текста

#### Футер с ценой:
- 💰 Крупный шрифт для цены (24pt black)
- 🎁 Иконки для каждого типа тарифа
- ✓ Круглая галочка для выбранной карточки

#### Эффекты и анимации:
- 📦 3D эффект с тенью при выборе
- 🎭 Легкий градиент на фоне выбранной карточки
- 🌊 Плавная spring анимация при нажатии
- 🔲 Scale эффект (1.02x) для выбранной карточки
- 💫 Градиентная обводка с цветом тарифа

## 🎬 Анимации

### 1. Spring анимация при выборе
```swift
.scaleEffect(isSelected ? 1.02 : 1.0)
.animation(.spring(response: 0.3, dampingFraction: 0.7), value: isSelected)
```

### 2. Press эффект (TierCardButtonStyle)
```swift
.scaleEffect(configuration.isPressed ? 0.98 : 1.0)
.animation(.easeInOut(duration: 0.15), value: configuration.isPressed)
```

---

## 🛠️ Новые компоненты

### 1. TierCardButtonStyle
Кастомный ButtonStyle для плавной анимации нажатия

### 2. RoundedCorner Shape
Позволяет закруглять отдельные углы (используется для бейджа)

```swift
extension View {
    func cornerRadius(_ radius: CGFloat, corners: UIRectCorner) -> some View {
        clipShape(RoundedCorner(radius: radius, corners: corners))
    }
}
```

## 📱 Пользовательский опыт

### Визуальная обратная связь:
1. **Hover** (на iPad/Mac): Легкое выделение
2. **Tap**: Scale down (0.98x)
3. **Selected**: Scale up (1.02x) + тень + градиентная обводка
4. **Transition**: Плавная spring анимация

### Информационная иерархия:
1. Иконка + Название тарифа (самое заметное)
2. Описание (вторичное)
3. Список функций (детали)
4. Цена/Статус (итог)
5. Галочка выбора (подтверждение)
