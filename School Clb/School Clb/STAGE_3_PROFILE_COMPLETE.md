## Созданные файлы:

### 1. **ProfileView.swift** ✅
Полнофункциональный экран профиля с:
- **Шапка профиля:**
  - Круглая аватарка (80pt)
  - Никнейм @username
  - Кнопка "Мой вайб" с эмодзи (открывает EmojiPickerView)
  - Прогресс-бар завершённых уроков
  - Дата последнего урока ("Акыркы сабак: 02 сент 2026")

- **Карточка подписки:**
  - Бесплатный аккаунт → показывает "Стать Премиум" (оранжевая кнопка)
  - Премиум аккаунт → показывает "Премиум до [дата]" (зелёный бейдж)

- **Секция АККАУНТ:**
  - Редактировать профиль → открывает EditProfileView
  - Обратиться за помощью → открывает email (support@schoolclub.kg)

- **Секция ДОКУМЕНТЫ:**
  - Публичная оферта (открывается в Safari)
  - Политика конфиденциальности (открывается в Safari)
  - О компании Скул Клаб (открывается в Safari)

- **Переключатель языка:**
  - KG / RU Picker
  - Сохраняется в UserDefaults

- **Выход:**
  - Кнопка "Чыгуу" (красный текст)
  - Версия 1.0.0 (серый текст внизу)

### 4. **ProfileViewModel.swift** ✅
ViewModel для управления профилем:
- **Текущая версия:** Работает с моковыми данными (без Firebase)
- **Функционал:**
  - Загрузка профиля пользователя
  - Обновление данных (ФИО, никнейм, район, школа)
  - Изменение вайба (эмодзи)
  - Загрузка аватарки
  - Изменение пароля
  - Выход из аккаунта
  - Расчёт прогресса (% завершённых уроков)

- **Модель данных:**
  ```swift

    district, school, isPremium, 
    premiumUntil, avatarURL, vibe
  }
  ```
