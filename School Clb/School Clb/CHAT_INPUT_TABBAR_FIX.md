# ✅ ИСПРАВЛЕНИЕ: Поле ввода скрыто за TabBar

## 🐛 Проблема
На скриншоте видно чат "Ортголики", но внизу **нет поля ввода** - только TabBar с кнопками "УРОКИ", "ДВИЖ", "БРЕЙК", "ЧАТ", "ПРОФИЛЬ".

## ✅ Решение

### 1. Скрыт TabBar в чате
```swift
.toolbar(.hidden, for: .tabBar) // Скрывает TabBar при входе в чат
```

### 2. Добавлен отступ для поля ввода
```swift
.padding(.bottom, 80) // Отступ на случай если TabBar виден
```
