# Исправление отступа поля ввода от TabBar

## 🐛 Проблема на скриншоте:

Поле ввода "Сообщение..." **прилипает к TabBar** снизу - между ними нет воздуха/отступа. Выглядит как одно целое.

```
┌────────────────────────┐
│ Сообщения...           │
│                        │
├────────────────────────┤ ← Нет отступа!
│ [Сообщение...    ] [➤] │ ← Поле ввода
├────────────────────────┤ ← Лепится!
│ УРОКИ | ДВИЖ | БРЕЙК   │ ← TabBar
└────────────────────────┘
```

---

## ✅ Решение:

Увеличены отступы снизу и добавлена тень для визуального разделения.

---

## Изменения в коде:

### MyGroupChatView.swift

**БЫЛО:**
```swift
.padding(.horizontal, 12)
.padding(.vertical, 8)
.padding(.bottom, 50) // Отступ для TabBar
.background(Color(uiColor: .systemBackground))
```

**СТАЛО:**
```swift
.padding(.horizontal, 12)
.padding(.vertical, 12) // Увеличено с 8 до 12
.padding(.bottom, 70) // Увеличено с 50 до 70!
.background(Color(uiColor: .systemBackground))
.shadow(color: Color.black.opacity(0.05), radius: 4, x: 0, y: -2) // Тень сверху
```

### AITeacherView.swift

**БЫЛО:**
```swift
.padding(12)
.background(Color(uiColor: .systemBackground))
.padding(.bottom, 50)
```

**СТАЛО:**
```swift
.padding(12)
.padding(.bottom, 70) // Увеличено с 50 до 70
.background(Color(uiColor: .systemBackground))
.shadow(color: Color.black.opacity(0.05), radius: 4, x: 0, y: -2)
```

---

## 🎨 Визуальные улучшения:

### 1. Отступ увеличен:
- **Было:** 50pt
- **Стало:** 70pt (+20pt воздуха)

### 2. Вертикальный padding:
- **Было:** 8pt
- **Стало:** 12pt (больше воздуха внутри)

### 3. Добавлена тень:
```swift
.shadow(color: Color.black.opacity(0.05), radius: 4, x: 0, y: -2)
```
- Легкая тень сверху поля ввода
- Визуально отделяет от контента
- Создает "парящий" эффект

## 📐 Размеры отступов:

| Элемент | Отступ | Назначение |
|---------|--------|------------|
| `.padding(.vertical, 12)` | 12pt | Внутренний отступ сверху/снизу контейнера |
| `.padding(.bottom, 70)` | 70pt | Отступ от TabBar |
| `.shadow(radius: 4, y: -2)` | - | Визуальное отделение |

**Итого:** Поле ввода поднято на **82pt** от нижнего края экрана (12 + 70)
