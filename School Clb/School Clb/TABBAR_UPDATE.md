# Обновление дизайна TabBar (Нижнее меню)

## Обзор изменений

Полностью переработан дизайн нижнего навигационного меню (TabBar) с современными эффектами, адаптацией к темной теме и улучшенной визуальной обратной связью.

---

## 🎨 Новый дизайн

### До:
- ❌ Белый фон (не адаптируется к темной теме)
- ❌ Круглый желтый индикатор выбора
- ❌ Серые иконки для неактивных табов
- ❌ Минимальные визуальные эффекты
- ❌ Обычная тень

### После:
- ✅ **Blur-эффект** (.ultraThinMaterial) - полупрозрачный фон
- ✅ **Градиентная линия** сверху (желто-оранжевая)
- ✅ **Округлый желтый бэдж** для активной кнопки (52x32pt)
- ✅ **Градиент на иконке** активного таба
- ✅ **Адаптивные цвета** для темной темы
- ✅ **Цветная тень** на активном табе
- ✅ **Spring анимации** при переключении
- ✅ **Scale эффект** при нажатии

---

## 📱 Компоненты

### 1. CustomTabBar (Контейнер)

#### Фон с Blur:
```swift
Rectangle()
    .fill(.ultraThinMaterial)  // Полупрозрачный материал
```

#### Градиентная линия сверху:
```swift
Rectangle()
    .fill(
        LinearGradient(
            colors: [
                Color(hex: "#FFE600").opacity(0.3),  // Желтый
                Color(hex: "#E84E1B").opacity(0.2)   // Оранжевый
            ],
            startPoint: .leading,
            endPoint: .trailing
        )
    )
    .frame(height: 1)
```

#### Тень:
- **Светлая тема:** Черная с opacity 0.08, radius: 12, y: -4
- **Темная тема:** Черная с opacity 0.3, radius: 12, y: -4

---

### 2. TabBarButton (Кнопка таба)

#### Активное состояние (isSelected = true):

**Фон:**
```swift
RoundedRectangle(cornerRadius: 16)
    .fill(
        LinearGradient(
            colors: [
                Color(hex: "#FFE600"),  // Желтый
                Color(hex: "#FFD000")   // Темнее желтый
            ],
            startPoint: .topLeading,
            endPoint: .bottomTrailing
        )
    )
    .frame(width: 52, height: 32)
```

**Тень на фоне:**
```swift
.shadow(
    color: Color(hex: "#FFE600").opacity(0.4),
    radius: 8,
    x: 0,
    y: 2
)
```

**Иконка:**
- Градиент от #1B2A6B до #2A3A7B
- Scale: 1.1x (увеличена)
- Font: 20pt, semibold

**Текст:**
- Font: 11pt, bold
- Color: .primary (адаптивный)

#### Неактивное состояние (isSelected = false):

**Иконка:**
- Color: .secondary (серая, адаптивная)
- Scale: 1.0x
- Font: 20pt, semibold

**Текст:**
- Font: 11pt, medium
- Color: .secondary

## 🎬 Анимации

### 1. Spring анимация при переключении:
```swift
withAnimation(.spring(response: 0.3, dampingFraction: 0.7)) {
    action()
}
```

### 2. Scale эффект при нажатии:
```swift
.scaleEffect(configuration.isPressed ? 0.92 : 1.0)
.animation(.easeInOut(duration: 0.15), value: configuration.isPressed)
```

### 3. Transition появления фона:
```swift
.transition(.scale.combined(with: .opacity))
```

### 4. Scale иконки:
```swift
.scaleEffect(isSelected ? 1.1 : 1.0)
```

## ✨ Особенности

### 1. Blur-эффект материала:
- Полупрозрачность
- Размытие контента позади
- Адаптация к светлой/темной теме
- Современный iOS-стиль

### 2. Градиентная линия-индикатор:
- Брендовые цвета (желтый + оранжевый)
- Визуально отделяет TabBar от контента
- Добавляет премиальности

### 3. Округлый бэдж вместо круга:
- Лучше подходит под иконки
- Более современный вид
- Градиент для глубины
- Цветная тень для эффекта свечения

### 4. Градиент на активной иконке:
- Темно-синий (брендовый цвет)
- Объем и глубина
- Лучшая видимость

### 5. Плавные анимации:
- Spring эффект при переключении
- Scale при нажатии
- Transition при появлении бэджа

## 🚀 Пользовательский опыт

### Визуальная обратная связь:
1. **Tap начало:** Scale down (0.92x)
2. **Tap конец:** Spring анимация
3. **Переключение:** 
   - Старый таб: fade out бэджа + scale down иконки
   - Новый таб: fade in бэджа + scale up иконки
4. **Активное состояние:**
   - Желтый бэдж с тенью
   - Градиент на иконке
   - Жирный текст

### Доступность:
- Адаптивные системные цвета
- Достаточный контраст в обеих темах
- Haptic feedback (опционально, можно добавить)
- Четкие иконки 20pt

## 🛠️ Новые компоненты

### TabBarButtonStyle
Кастомный ButtonStyle для плавной анимации нажатия

```swift
struct TabBarButtonStyle: ButtonStyle {
    func makeBody(configuration: Configuration) -> some View {
        configuration.label
            .scaleEffect(configuration.isPressed ? 0.92 : 1.0)
            .animation(.easeInOut(duration: 0.15), value: configuration.isPressed)
    }
}
```
