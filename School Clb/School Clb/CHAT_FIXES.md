# Исправления вкладки ЧАТ

## Проблемы на скриншоте:

1. ❌ Текст "ИИ-ПРЕПОДАВАТЕЛЬ ПОМОЩНИК ЕЛЬ" разбивается на строки
2. ❌ Текст "ЛИЧНЫЕ ЧАТЫ СООБЩЕНИЯ" разбивается некрасиво
3. ❌ Тариф STUDENT не открывает доступ к "Моя группа"

---

## ✅ Исправления:

### 1. Убраны subtitle из названий карточек

**Было:**
```swift
title: "ФОРУМ"
subtitle: "СКУЛ КЛАБ"
```

**Стало:**
```swift
title: "ФОРУМ СКУЛ КЛАБ"
subtitle: ""
```

**Результат:** Текст теперь на одной строке, не разбивается.

---

### 2. Обновлена структура ChatSectionCard

**Было:**
```swift
HStack(spacing: 6) {
    Text(title)
        .font(.system(size: 14, weight: .bold, design: .rounded))
        .foregroundColor(isLocked ? .secondary : Color(hex: color))
    
    Text(subtitle)
        .font(.system(size: 14, weight: .bold, design: .rounded))
        .foregroundColor(isLocked ? .secondary : .primary)
}
```

**Стало:**
```swift
Text(title)
    .font(.system(size: 15, weight: .bold, design: .rounded))
    .foregroundColor(isLocked ? .secondary : .primary)
    .lineLimit(1)
```

**Результат:** 
- Одна строка для заголовка
- Увеличен шрифт до 15pt
- Добавлен `lineLimit(1)` для обрезки длинных названий
- Единообразный цвет `.primary` для всех заголовков

---

### 3. Исправлен тариф по умолчанию в ProfileViewModel

**Было:**
```swift
subscriptionTier: .base  // Default subscription tier
```

**Стало:**
```swift
subscriptionTier: .student  // СТУДЕНТ получает STUDENT тариф
```

**Результат:** Теперь при запуске приложения пользователь имеет тариф STUDENT и видит:
- ✅ "МОЯ ГРУППА ОРТГОЛИКИ" (открыта)
- 🔒 "ФОРУМ СКУЛ КЛАБ" (закрыт)
- 🔒 "ИИ-ПРЕПОДАВАТЕЛЬ" (закрыт)
- 🔒 "ЛИЧНЫЕ ЧАТЫ" (закрыт)

---
