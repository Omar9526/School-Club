## Единый стиль пузырей:

### Для ВСЕХ чатов:
- **Моё сообщение**: справа, цветной пузырь (accent color), белый текст
- **Чужое сообщение**: слева, светлый пузырь (.systemBackground), тёмный текст
- **Радиус**: 16pt для текста, 12pt для изображений
- **Отступы**: 12pt внутри пузыря
- **Шрифт**: .system(size: 15, weight: .regular, design: .rounded)
- **Максимальная ширина**: 280pt для пузыря, 250pt для изображений
- **Время**: под пузырём, .system(size: 11, weight: .regular), .secondary

### Особенности по типам:
- **Групповые и ИИ**: показывают имя автора и аватар слева для чужих сообщений
- **Личные**: без имени и аватара (1-на-1 диалог)
- **Форум**: ответы показывают имя автора и кнопку лайка

## Поддержка вложений:

### Типы файлов:
1. **Фото и видео** — через PhotosPicker (.images, .videos)
2. **Документы** — через FileImporter (.pdf, .plainText, .png, .jpeg)

### Отображение вложений в пузыре:
- **Изображение**: Image с .scaledToFit, radius 12
- **Видео**: превью (чёрный прямоугольник с иконкой play)
- **Документ**: иконка doc.fill + имя файла

### Превью перед отправкой:
- Показывается над полем ввода
- Кнопка крестика для удаления
- Фон: .secondarySystemBackground
