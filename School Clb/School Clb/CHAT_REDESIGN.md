# Переработка вкладки ЧАТ

## Обзор изменений

Полностью переработана вкладка "ЧАТ" в приложении School Club. Вместо двух табов (Форум/ИИ) создан **главный хаб с 4 разделами чатов** с разными уровнями доступа по тарифам.

---

## 🎯 Новая структура

### ДО:
```
ЧАТ
├── Таб 1: ФОРУМ / ЧАТ
└── Таб 2: ИИ ПРЕПОДАВАТЕЛЬ
```

### ПОСЛЕ:
```
ЧАТ (Главный хаб)
├── 1. МОЯ ГРУППА "ОРТГОЛИКИ" (STUDENT/PRO)
│   └── или "СТАТЬ СТУДЕНТОМ" (для BASE)
├── 2. ФОРУМ СКУЛ КЛАБ (PRO)
├── 3. ИИ-ПРЕПОДАВАТЕЛЬ (PRO)
└── 4. ЛИЧНЫЕ ЧАТЫ (PRO)
```

---

## 📱 Разделы чатов

### 1️⃣ МОЯ ГРУППА "ОРТГОЛИКИ"
**Доступ:** STUDENT или PRO

**Для студентов (STUDENT/PRO):**
- Групповой чат с одногруппниками
- Badge с количеством новых сообщений (+30)
- Возможность отправлять текстовые сообщения
- Видны никнеймы и аватары участников

**Для не-студентов (BASE):**
- Показывается карточка "СТАТЬ СТУДЕНТОМ SCHOOL CLUB"
- Кнопка-ссылка на сайт для записи
- Описание преимуществ группового чата

**Файлы:**
- `MyGroupChatView.swift` - экран группового чата
- `GroupMessage` - модель сообщения группы

---

### 2️⃣ ФОРУМ СКУЛ КЛАБ
**Доступ:** PRO

**Функционал:**
- Задавать вопросы по предметам
- Прикреплять фото заданий
- Получать ответы от других студентов
- Ставить лайки полезным ответам
- Модерация контента

**Разделы форума:**
- Математика
- Физика
- Химия
- Другие предметы

**Файлы:**
- `ForumChatView.swift` - список вопросов
- `AskQuestionView.swift` - создание вопроса
- `PostDetailView.swift` - детальный просмотр вопроса с ответами

---

### 3️⃣ ИИ-ПРЕПОДАВАТЕЛЬ
**Доступ:** PRO

**Функционал:**
- Чат с искусственным интеллектом
- Помощь в решении задач
- Можно прикреплять фото заданий
- AI объясняет решение пошагово

**Файлы:**
- `AITeacherView.swift` - чат с AI
- `AIMessage` - модель сообщения

---

### 4️⃣ ЛИЧНЫЕ ЧАТЫ
**Доступ:** PRO

**Функционал:**
- Личная переписка с друзьями
- Чаты с учителями (отмечены галочкой)
- Только с подтвержденными контактами
- Только студенты и администраторы School Club
- Поиск по чатам
- Badge с непрочитанными сообщениями

**Файлы:**
- `PrivateChatsView.swift` - список чатов
- `PrivateChatDetailView.swift` - детальный чат
- `PrivateChat` - модель чата
- `PrivateMessage` - модель сообщения

---

## 🔒 Система доступа

### BASE (Бесплатный тариф):
- ❌ Моя группа → Показывается "Стать студентом"
- ❌ Форум → Заблокирован (замочек)
- ❌ ИИ-преподаватель → Заблокирован (замочек)
- ❌ Личные чаты → Заблокированы (замочек)

### STUDENT (Для студентов School Club):
- ✅ Моя группа → Доступ открыт
- ❌ Форум → Заблокирован
- ❌ ИИ-преподаватель → Заблокирован
- ❌ Личные чаты → Заблокированы

### PRO (Платная подписка):
- ✅ Моя группа → Доступ открыт
- ✅ Форум → Доступ открыт
- ✅ ИИ-преподаватель → Доступ открыт
- ✅ Личные чаты → Доступ открыт

**Проверка в коде:**
```swift
// Моя группа
if user.hasStudentAccess { ... } // STUDENT или PRO

// Остальные разделы
if user.hasProAccess { ... } // Только PRO
```

Каждая карточка раздела имеет:

- 🎨 **Иконка** в градиентном круге

- ➡️ **Стрелка** для перехода
- 💫 **Тень** для глубины

| Раздел | Цвет | Иконка |
|--------|------|--------|
| Моя группа | #2196F3 (синий) | person.3.fill |
| Форум | #E84E1B (оранжевый) | bubble.left.and.bubble.right.fill |
| ИИ | #9C27B0 (фиолетовый) | brain.head.profile |
| Личные чаты | #34C759 (зеленый) | message.fill |

### Состояния:
- **Активна:** Полная непрозрачность, тень
- **Заблокирована:** 60% прозрачность, замочек, нельзя нажать

## 📊 Навигация

### Главный экран ChatView:
```
ScrollView
└── VStack (4 карточки)
    ├── ChatSectionCard (Моя группа) → MyGroupChatView
    │   или BecomeStudentCard → Ссылка на сайт
    ├── ChatSectionCard (Форум) → ForumChatView
    ├── ChatSectionCard (ИИ) → AITeacherView
    └── ChatSectionCard (Личные) → PrivateChatsView
```

### Проверка доступа:
```swift
// В ChatView
if profileViewModel.user?.hasStudentAccess == true {
    // Показать "Моя группа"
} else {
    // Показать "Стать студентом"
}

// Для других разделов
isLocked: !(profileViewModel.user?.hasProAccess ?? false)
```
