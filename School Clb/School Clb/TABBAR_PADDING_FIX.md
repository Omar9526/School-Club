# Исправление: Таб-бар перекрывает контент

## Проблема
Нижний таб-бар перекрывал последние карточки на всех экранах — до последних элементов было невозможно доскроллить.

## Решение
Добавлен нижний отступ `.padding(.bottom, 100)` во все основные экраны с ScrollView.

## Изменённые файлы:

### 1. ✅ MovementView.swift (Курс)
```swift
.padding(20)
.padding(.bottom, 100) // ✅ Отступ снизу, чтобы контент не заезжал под таб-бар
```
**Результат:** Последняя карточка "ВАШЕ РАСПИСАНИЕ" теперь полностью видна над таб-баром.

### 2. ✅ BreakView.swift (Брейк)
```swift
.padding(20)
.padding(.bottom, 100) // ✅ Отступ снизу, чтобы контент не заезжал под таб-бар
```
**Результат:** Секция "ГРУППОВЫЕ ИГРЫ" полностью доступна для скролла.

### 3. ✅ LessonsView.swift (Уроки)
```swift
.padding(20)
.padding(.bottom, 100) // ✅ Отступ снизу, чтобы контент не заезжал под таб-бар
```
**Результат:** Последние курсы в списке не перекрываются таб-баром.

### 4. ✅ ProfileView.swift (Профиль)
```swift
.padding(.vertical, 20)
.padding(.bottom, 100) // ✅ Отступ снизу, чтобы контент не заезжал под таб-бар
```
**Результат:** Текст версии и кнопка выхода полностью видны.

### 5. ✅ ChatView.swift (Чат)
```swift
.padding(20)
.padding(.bottom, 100) // ✅ Отступ снизу, чтобы контент не заезжал под таб-бар
```
**Результат:** Карточка "ЛИЧНЫЕ ЧАТЫ" полностью доступна.

## Проверка
Теперь на всех 5 вкладках:
- ✅ Уроки
- ✅ Курс (бывший Движ)
- ✅ Брейк
- ✅ Чат
- ✅ Профиль

Последние элементы не перекрываются таб-баром и полностью доступны для просмотра и взаимодействия.

## Единообразие
Во всех экранах используется одинаковый отступ `100pt`, что обеспечивает комфортное расстояние между последним элементом контента и таб-баром на всех устройствах.
