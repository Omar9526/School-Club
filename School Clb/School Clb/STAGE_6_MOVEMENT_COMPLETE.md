## Система баллов:

✅ **Получение баллов:**
- Вход в приложение: +1% активности
- Правильный ответ в тесте: +5 баллов (базовые)
- Викторина (с множителем сложности)
- Дуэль (победа): +10 баллов за раунд
- Ежедневный вызов: +10 баллов

✅ **Трата баллов:**
- Партнёры: 50 баллов
- Отдать группе: 20 баллов
- Розыгрыш: 100 баллов
- Мерч: 150 баллов
- Благотворительность: 30 баллов
- Скидка на подписку: 200 баллов

## Firestore структура (для будущей интеграции):

```
users/{userId}
  - points: Int (текущий баланс)
  - activityRating: Double (0.0-1.0)
  - lastActiveDate: Timestamp
  - totalLogins: Int

announcements/{announcementId}
  - type: String ("award" | "birthday" | "news" | "charity" | "volunteer" | "partner")
  - title: String
  - body: String
  - imageURL: String?
  - createdAt: Timestamp
  - priority: Int

users/{userId}/pointsHistory/{transactionId}
  - type: "earn" | "spend"
  - amount: Int
  - category: String?
  - description: String
  - createdAt: Timestamp

leaderboard/{period}
  - userId: String
  - points: Int
  - rank: Int
  - updatedAt: Timestamp
```
