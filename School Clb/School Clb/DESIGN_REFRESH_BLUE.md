### 2. ✅ Кнопки ответов А/Б/В/Г — синий кружок с белой буквой

#### TestView.swift — AnswerButton

**Кружок с буквой (обычное состояние):**
```swift
// Было:
.foregroundColor(Color(hex: "#1B2A6B"))  // Синяя буква
.fill(Color(hex: "#F5F5F5"))  // Светло-серый фон

// Стало:
.foregroundColor(.white)  // Белая буква
.fill(Color(hex: "#1B2A6B"))  // Синий фон
```

**Фон кнопки при выборе:**
```swift
// Было:
.background(Color(hex: "#FFE600").opacity(0.2))  // Жёлтый

// Стало:
.background(Color(hex: "#1B2A6B").opacity(0.05))  // Синий лёгкий
```

**Рамка кнопки:**
```swift
// Было:
.stroke(Color.gray.opacity(0.3))  // Едва заметная серая

// Стало:
.stroke(Color(hex: "#1B2A6B").opacity(0.2))  // Заметная синяя
```

**Рамка при выборе:**
```swift
// Было:
.stroke(Color(hex: "#FFE600"))  // Жёлтая

// Стало:
.stroke(Color(hex: "#1B2A6B"))  // Синяя
```

**Состояния верно/неверно:**
- ✅ Зелёный и красный сохранены без изменений
- Буквы в состоянии верно/неверно — белые на цветном фоне
