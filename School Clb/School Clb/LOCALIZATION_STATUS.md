## 🎯 Как работает система

1. **Выбор языка:** Пользователь переключает KG/RU в ProfileView
2. **Сохранение:** `AppLanguage.current = "KG"` или `"RU"`
3. **Уведомление:** Отправляется `.languageChanged` notification
4. **Обновление:** Все экраны с `.onReceive` автоматически перерисовываются
5. **Отображение:** Все тексты через `L10n.*` показывают правильный язык

## 📱 Примеры использования

### Простой текст
```swift
Text(L10n.profile)
```

### С переменными
```swift
Text("\(level.freeLessons) \(L10n.free)")
```

### Заголовок навигации
```swift
.navigationTitle(L10n.lessons)
```

### Выбор по языку в компонентах
```swift
var displayTitle: String {
    AppLanguage.current == "RU" ? course.titleRu : course.titleKg
}
```

## 🔧 Технические детали

### Файлы системы
- `Localizable.swift` - все переводы
- `AppLanguage.current` - текущий язык (KG/RU)
- `.languageChanged` - notification для обновления UI

### Обязательные модификаторы
```swift
.id(refreshView)
.onReceive(NotificationCenter.default.publisher(for: .languageChanged)) { _ in
    refreshView.toggle()
}
```
