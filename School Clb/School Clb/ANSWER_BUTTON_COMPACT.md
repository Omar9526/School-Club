# Оптимизация размера кнопок ответов в тестах

## Изменённый компонент: AnswerButton

### Было:
```swift
HStack(spacing: 16) {
    Text(letter)
        .font(.system(size: 18, weight: .bold))
        .frame(width: 40, height: 40)
    
    Text(text)
        .font(.system(size: 15, weight: .regular))
    
    Image(systemName: "checkmark.circle.fill")
}
.padding(16)
```

### Стало:
```swift
HStack(spacing: 12) {  // ✅ 16 → 12
    Text(letter)
        .font(.system(size: 16, weight: .bold))  // ✅ 18 → 16
        .frame(width: 32, height: 32)  // ✅ 40x40 → 32x32
    
    Text(text)
        .font(.system(size: 14, weight: .regular))  // ✅ 15 → 14
    
    Image(systemName: "checkmark.circle.fill")
        .font(.system(size: 18))  // ✅ Добавлен размер иконки
}
.padding(10)  // ✅ 16 → 10
```

## Изменения в файлах:

### 1. ✅ TestView.swift — AnswerButton
- **Padding**: `16` → `10` (уменьшены вертикальные/горизонтальные отступы)
- **Кружок с буквой**: `40x40` → `32x32`
- **Шрифт буквы**: `18` → `16`
- **HStack spacing**: `16` → `12`
- **Текст ответа**: `15` → `14`
- **Иконка**: добавлен `.font(.system(size: 18))` для контроля размера

### 2. ✅ TestView.swift — VStack с кнопками
- **Spacing между кнопками**: `12` → `8`

### 3. ✅ PracticeTestView.swift — VStack с кнопками
- **Spacing между кнопками**: `12` → `8`

## Результат:

### Компактность:
- ✅ Кнопки занимают **~20% меньше места** по вертикали
- ✅ Между кнопками меньше пустого пространства
- ✅ На экране помещается больше вопроса/контекста

### Удобство:
- ✅ Кнопки остаются достаточно большими для тапа (~48pt высота)
- ✅ Текст читабельный (14pt)
- ✅ Визуальная иерархия сохранена

### Дизайн:
- ✅ Все цвета сохранены (зелёный/красный/жёлтый)
- ✅ Логика состояний работает (выбран/верный/неверный)
- ✅ Дизайн-система не нарушена
