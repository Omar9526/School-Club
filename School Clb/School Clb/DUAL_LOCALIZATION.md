# 🌐 Двойная система локализации School Club

## 📖 Концепция

В School Club используется **двойная система локализации**:

### 1. Язык интерфейса (UI Language)
Язык кнопок, меню, навигации - выбирается в **ProfileView**

### 2. Язык контента (Content Language)  
Язык внутри уроков, видео, тестов - зависит от **выбранного курса**

---

## 🎯 Как это работает

### Сценарий 1: Студент НЕ знает русский

**Шаг 1:** Выбирает язык **КЫРГЫЗСКИЙ** в ProfileView
```
✅ Интерфейс: Сабактар, Деңгээлдер, Профайл...
```

**Шаг 2:** Выбирает курс **"ЖРТ кыргыз тилинде"**
```swift
Course(contentLanguage: "KG")
```

**Шаг 3:** Открывает уроки
```
✅ Названия уроков: "Сабак 1", "Сабак 2"...
✅ Пробный тест: "СЫНАМЫК ТЕСТ №1"
✅ Описание: "Учурдагы деңгээлиңизди баалоо үчүн..."
✅ Видео: на кыргызском языке
✅ Тесты: на кыргызском языке
```

### Сценарий 2: Студент НЕ знает кыргызский

**Шаг 1:** Выбирает язык **РУССКИЙ** в ProfileView
```
✅ Интерфейс: Уроки, Уровни, Профиль...
```

**Шаг 2:** Выбирает курс **"ОРТ на русском языке"**
```swift
Course(contentLanguage: "RU")
```

**Шаг 3:** Открывает уроки
```
✅ Названия уроков: "Урок 1", "Урок 2"...
✅ Пробный тест: "ПРОБНЫЙ ТЕСТ №1"
✅ Описание: "Пробный тест для оценки вашего текущего уровня"
✅ Видео: на русском языке
✅ Тесты: на русском языке
```

## 🔧 Техническая реализация

### 1. Модель Course

```swift
struct Course {
    let titleKg: String         // "ЖРТ кыргыз тилинде"
    let titleRu: String         // "ОРТ на кыргызском"
    let contentLanguage: String // "KG" или "RU" ⭐ ВАЖНО!
}
```

### 2. Модель Lesson

```swift
struct Lesson {
    let titleKg: String        // "Сабак 1"
    let titleRu: String        // "Урок 1"
    let descriptionKg: String  // "Сабактын сүрөттөмөсү..."
    let descriptionRu: String  // "Описание урока..."
}
```

### 3. Отображение контента

```swift
// В LessonCard
var displayTitle: String {
    guard let course = viewModel.selectedCourse else { return lesson.title }
    return course.contentLanguage == "RU" ? lesson.titleRu : lesson.titleKg
}
```

## 🚀 Преимущества этой системы

1. ✅ **Гибкость**: Студент может менять язык интерфейса независимо от контента
2. ✅ **Удобство**: Каждый студент учится на своем языке
3. ✅ **Масштабируемость**: Легко добавлять новые курсы на разных языках
4. ✅ **Понятность**: Студент сразу видит какой контент в каком курсе
