## Созданные файлы:

### 1. **LessonsViewModel.swift** ✅
ViewModel для управления уроками:
- **Модели данных:**
  - `Course` - курс (ЖРТ, ОРТ, ХИМБИО и т.д.)
  - `CourseLevel` - уровень курса (А, В, С деңгээли)
  - `Lesson` - урок с видео и тестами
  - `Question` - вопрос теста (text или image)
  
- **Функционал:**
  - Загрузка курсов по категориям
  - Загрузка уровней для курса
  - Загрузка уроков для уровня
  - Загрузка вопросов для теста
  - Проверка доступа (бесплатно/премиум)
  - Сохранение прогресса
  
- **Текущая версия:** Mock данные (готово к Firebase)

### 2. **LessonsView.swift** ✅
Главный экран уроков (страница 1):
- **Табы категорий:**
  - SC | ЖРТ жөнүндө

- **Список курсов:**

  - Башка предметтик тесттер
  
- **UI:** Карточки с иконками и описанием

### 3. **CourseLevelsView.swift** ✅
Экран уровней курса (страница 2):
- **3 уровня:**
  - А деңгээли (0-120 балл) - 24 урока: 6 бесплатно + 18 премиум
  - В деңгээли (121-170 балл) - аналогично
  - С деңгээли (171-245 балл) - аналогично
  
- **Информация на карточке:**
  - Диапазон баллов
  - Количество уроков
  - Количество бесплатных/премиум уроков
  - Прогресс-бар (заглушка 0%)
  
- **Защита:** Замок на премиум уроках

### 4. **LessonListView.swift** ✅
Список уроков в уровне (страница 3):
- **СЫНАМЫК ТЕСТ №1:**
  - Отдельная карточка вверху
  - Яркий дизайн со звездой
  - Всегда бесплатный
  
- **Список уроков (Сабак 1, 2, 3...):**
  - Каждый урок имеет 3 кнопки:

    3. **ТЕСТ — өз алдынча иштөө** - жёлтая
  
- **Защита контента:**
  - Замок на платных уроках
  - Alert при попытке открыть без премиума

### 5. **VideoLessonView.swift** ✅
Видеоплеер (страница 4):
- **AVPlayer:** Встроенный iOS плеер
- **Функционал:**
  - Воспроизведение видео
  - Полноэкранный режим
  - Зум жестом (MagnificationGesture 1.0x - 3.0x)
  - Защита от скриншотов (детекция)
  
- **UI:**
  - Видео вверху (250pt высота)
  - Описание урока внизу (ScrollView)
  - Предупреждение о защите контента
  
- **Типы видео:**
  - Теоретический урок (синяя иконка)
  - Видеоразбор теста (оранжевая иконка)

### 6. **TestView.swift** ✅
Экран теста (страница 5):
- **Прогресс-бар:** Вопрос X из N, процент
- **Типы вопросов:**
  - **Image** - для математики/химии/биологии (AsyncImage)
  - **Text** - для текстовых предметов
  
- **4 варианта ответа:** А, Б, В, Г
- **Логика:**
  1. Выбор ответа
  2. Кнопка "Проверить"
  3. Подсветка: зелёный (правильно) / красный (неправильно)
  4. Кнопка "Следующий вопрос"
  
- **Результаты:**
  - Процент правильных ответов
  - Эмодзи успеха/неудачи
  - Кнопка видеоразбора (если есть)

### 7. **PracticeTestView.swift** ✅
Пробный тест (СЫНАМЫК ТЕСТ):
- **Аналогично TestView** но с расширенными результатами:
  - Общий процент
  - **Детальный разбор каждого вопроса:**
    - Номер вопроса
    - Текст вопроса
    - Ваш ответ (если неправильно)
    - Правильный ответ
    - Зелёная/красная подсветка карточек
  
- **Видеоразбор:** Ссылка на полный видеоразбор теста
- **UI:** Жёлтый акцент (тема пробного теста)

## Структура навигации:

```
LessonsView (табы + курсы)
  ↓
CourseLevelsView (3 уровня)
  ↓
LessonListView (СЫНАМЫК ТЕСТ + список уроков)
  ↓
  ├─→ VideoLessonView (теория)
  ├─→ TestView (тест с видеоразбором)
  ├─→ TestView (тест самостоятельно)
  └─→ PracticeTestView (пробный тест)
```

## Firestore структура (для будущей интеграции):

```
courses/{courseId}
  - title: String
  - titleKg: String
  - titleRu: String
  - order: Int
  - category: String

courses/{courseId}/levels/{levelId}
  - title: String
  - minScore: Int
  - maxScore: Int
  - order: Int
  - totalLessons: Int
  - freeLessons: Int

courses/{courseId}/levels/{levelId}/lessons/{lessonId}
  - title: String
  - order: Int
  - isFree: Bool
  - isPracticeTest: Bool
  - videoURL: String (Firebase Storage)
  - videoAnalysisURL: String (optional)
  - description: String
  - subject: "math" | "chemistry" | "biology" | "text"

courses/{courseId}/levels/{levelId}/lessons/{lessonId}/questions/{questionId}
  - order: Int
  - type: "image" | "text"
  - questionText: String (optional)
  - questionImageURL: String (optional, Firebase Storage)
  - optionA: String
  - optionB: String
  - optionC: String
  - optionD: String
  - correctAnswer: "A" | "B" | "C" | "D"

users/{userId}/progress/{lessonId}
  - completedAt: Timestamp
  - score: Int
  - totalQuestions: Int
  - percentage: Int
```
