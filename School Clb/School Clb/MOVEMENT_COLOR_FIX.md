# Исправление цветов экрана "Курс"

## Изменения в MovementView.swift

### 1. ✅ Карточка "МОИ БАЛЛЫ" — убран жёлтый фон/обводка

**Было:**
- Жёлтый градиентный фон: `Color(hex: "#FFE600").opacity(0.15)` → `0.05`
- Жёлтая обводка: `.stroke(Color(hex: "#FFE600").opacity(0.3), lineWidth: 2)`
- Иконка person.fill жёлтая: `Color(hex: "#FFE600")`

**Стало:**
- Чистый белый фон: `Color.white`
- Лёгкая тень: `.shadow(color: Color.black.opacity(0.05), radius: 8, x: 0, y: 2)`
- Иконка person.fill синяя: `Color(hex: "#1B2A6B")`
- Звезда и число баллов остались (звезда жёлтая, число синее)

```swift
.background(Color.white)
.cornerRadius(16)
.shadow(color: Color.black.opacity(0.05), radius: 8, x: 0, y: 2)
```

### 2. ✅ Экран "Отметки группы" — оценки жёлтые → оранжевые

**Было:**
```swift
Text(student.grade)
    .foregroundColor(Color(hex: "#FFE600")) // Жёлтый
```

**Стало:**
```swift
Text(student.grade)
    .foregroundColor(Color(hex: "#E84E1B")) // Оранжевый
```

**Результат:**
- Оценки (5.0, 4.8, 4.5...) теперь оранжевые `#E84E1B`
- Имена студентов остались синими `#1B2A6B`
- "Средний балл" остался серым `.gray`
- Цвет оценок совпадает с результатами тестов (8/10, 9/10)

## Дизайн-система сохранена:
- Синий `#1B2A6B` — основной цвет
- Оранжевый `#E84E1B` — акцент для баллов/оценок
- Жёлтый `#FFE600` — только для звёзд (награды)
- Белый фон + лёгкие тени для карточек

## Визуальная согласованность:
Теперь все числовые показатели (оценки, результаты тестов) имеют единый оранжевый цвет, что делает интерфейс более целостным.
